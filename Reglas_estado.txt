Reglas de cambios de estado
una cita solo puede ser modificada si este sigue con estado pendiente
no se puede modificar cuando ya esta aprobado 
solo se puede iniciar una cita que este confirmada 
se puede cancelar una cita si esta confirmada o pendiente
una vez este pendiente puede cancelar solo el solicitante
una vez rechazado solo puede cancelar el solicitante
una vez confirmado solo pueden cancelar ambos
solo puede finalizarse la cita si es que esta confirmado
para iniciar debe estar confirmado
